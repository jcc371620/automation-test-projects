# Automation Projects

一个集合多种自动化测试示例与框架的项目仓库，涵盖 API 测试、Web 测试、移动端测试和性能测试。

---

## 📂 项目结构

- **apitest_pytest_requests/** – API 测试示例（Requests + Pytest）  
- **apptest_pytest_appium/** – 移动端 App 测试示例（Appium + Pytest）  
- **webtest_pytest_playwright/** – Web 测试示例（Playwright + Pytest）  
- **test_automation_framework/** – 从零搭建的全栈自动化工程
- **postman/** – Postman 接口测试集合  
- **jmeter/** – 性能测试脚本  
- **python/** – Python 脚本与练习  
- **java/**, **c++/** – 语言练习与工具  
- **assets/** – 测试资源文件  
- **.vscode/** – VS Code 配置  
- **report.html / index.html** – 测试报告与结果展示  
- **google_result.png** – 示例截图

---

## 🛠 技术栈

- **编程语言**: Python, Java, C++  
- **测试框架**: Pytest, Playwright, Appium  
- **性能测试**: JMeter  
- **接口测试**: Postman, Python Requests  
- **工具**: VS Code
